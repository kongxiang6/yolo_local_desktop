YOLO训练工具 - 小白使用说明



一、先记住四句话

1. 不要只拿一个 exe 单独运行

2. 第一次打开后，先点“在线一键配置环境”

3. 看不懂的参数先别改，先用默认值或推荐预设

4. 想看所有参数的详细中文解释，请打开同目录里的“USER_GUIDE.txt”


二、第一次怎么用

1. 先把整个压缩包完整解压

2. 双击打开“YOLO训练工具.exe”

3. 先点“在线一键配置环境”

4. 等左侧日志提示完成

5. 再去做训练、验证、预测、跟踪或导出

说明：现在发出去的压缩包默认可能不自带 runtime，这是正常的；第一次点“在线一键配置环境”后，软件会自动创建自己的内置环境


三、几个按钮是什么意思

1. 在线一键配置环境：只会检查并配置软件内置 runtime，不会往别的路径里装环境
2. 检查当前环境：检查的是软件现在这套内置环境能不能直接用
3. 如果检查失败或还没装好，就先点“在线一键配置环境”
4. 下载慢时，程序会先测速几个国内镜像，再优先走更快的源；下过的文件也会自动复用缓存


四、最稳的用法

1. 先配好环境

2. 先用推荐预设

3. 看不懂的参数先别改

4. 最常改的一般只有 epochs、batch、imgsz、conf


五、出错了怎么办

1. 先看左侧日志

2. 点“打开日志”

3. 把报错截图发回来
